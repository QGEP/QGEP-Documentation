6.5. Datenimport

Sobald die Übereinstimmung erreicht ist, können die Untersuchungsdaten mit dem Import Knopf importiert werden. Es wird ein Check durchgeführt und der Import gestopt, wenn ein Problem gefunden wird. Siehe Problemlösungen.

Wenn alle selektierten Haltungen und Untersuchungsberichte korrekt sind, werden die Daten im QGEP-Datenmodell in der VSA-DSS KEK Erweiterung importiert.

Bemerkung

Achtung: Importierte Daten können im Moment nicht einfach wieder gelöscht werden, es gibt kein „Wiederrufen/Zurück/Undo“. Wenn die Daten noch einmal importiert werden, werden sie noch einmal eingefügt und sind dann doppelt vorhanden.
