6.4. Problemlösungen¶

6.4.1. Probleme mit Längenangabe¶

Wenn ein Untersuchungsbericht eine grössere Länge hat als die Haltung im Plan, erscheint eine Warnung. Dies sollte in den Daten festgelegt sein oder kann in den Untersuchungsdaten erzwungen werden, indem die entsprechende Spalte markiert wird

6.4.2. Ein Kanal/Bericht in den Inspektionsdaten entspricht mehreren Kanälen in den QGEP-Daten¶

Ein Bericht in den Kanalfernsehdaten kann mehrere Haltungen/Kanäle in den QGEP-Daten entsprechen. In diesem Fall können bis zu 3 Haltungen im Widget gewählt werden. Die Beobachtungslängen werden entsprechend angepasst.

6.4.3. Mehrere Berichte in den Kanalfernsehdaten entsprechen einer einzelnen Haltung in QGEP¶

Ein Bericht in den Kanalfernsehdaten kann mehrere Haltungen/Kanäle in den QGEP-Daten entsprechen. In diesem Fall kann die Checkbox „Reuse previous channel“ selektiert werden. Die Beobachtungslängen werden entsprechend angepasst.
