2.2.2. Abwasserbauwerke digitalisieren¶

2.2.2.1. Allgemeines¶

Die Option Letzte Attributwerteingabe wiederholen (Einstellungen -> Optionen -> Digitalisierung) deaktivieren, weil der zweite neue Datensatz nicht mehr gesichert werden kann. Dieser Bug wird spätestens mit QGIS 3.0 behoben sein.

QGEP hat einen Assistenten, um Schächte und Sonderbauwerke zu erfassen.

Wählen Sie den Wizard Knopf und dann klicken Sie auf Start Data Entry und wählen dann Wastewater Structure im herunterzieh Menu

2.2.2.2. Digitalisieren¶

Jetzt ändert sich der Cursor zum Digitalisierungssymbol und Sie können den Standort des neuen Punktelementes festlegen.

Es öffnet das vw_gep_wastewater_structure Formular und Sie können beginnen, Daten in den General Tab einzutragen:

Wähle den Abwasserbauwerk-Typ (vorgewählt ist ‘Normschacht’):

  • Einleitstelle

  • Versickerungsanlage

  • Normschacht

  • Spezialbauwerk

Fügen Sie dann die Bauwerks-Bezeichnung hinzu (dies ist die Bezeichnung, welche auf den Plänen angezeigt werden wird). Die Bezeichnungen von Deckel und Abwasserknoten werden als Voreinstellung gleich wie die Bauwerksbezeichnung gesetzt.

Für Kanalschächte und Sonderbauwerke wählen Sie dann die Funktion (in manhole_function oder in special structure_function). Es wird dann automatisch angezeigt oder in Zukunft versteckt, abhängig vom ws_type)

Fügen Sie weitere Attributwerte im General**Tab hinzu. Sie können auch Attributwerte in anderen Tabs hinzufügen (Deckel, Abwasserbauwerk, Normschacht, Abwasserknoten). In den Tabs Deckels, Bauwerkteile, Abwasserknotens und Erhaltungsereignisse können keine neuen Datensätze hinzugefügt werden, weil zuerst der Abwasserbauwerk-Datensatz in der Datenbank gesichert werden muss. Sie können auch die **Aktionen aus dem gleichen Grund noch nicht benutzen. Zuerst den Datensatz sichern. Jetzt bitte OK klicken.

Speichern Sie die Informationen dieser Ebene indem Sie den data entry Wizard stoppen

Sie können Ihr Punktobjekt bearbeiten, indem Sie den Bearbeitungsmodus auswählen und Sie dann mit dem Info Tool auf das Objekt klicken, welches Sie bearbeiten möchten. Wenn Sie den Bearbeitungsmodus nicht auswählen, können Sie nur die existierenden Daten anschauen.

Für detailliertere Informationen über das Bearbeiten lesen die das Kapitel Bearbeiten von bestehenden Daten.

2.2.2.3. Weitere Attribute und Klassen¶

Wenn ein Punktobjekt digitalisiert wird, findet eine Reihe von Schritten im Hintergrund der QGIS Datenbank statt:

  1. ein neues Objekt Abwasserbauwerk wird hinzugefügt

  2. ein neues Objekt der entsprechenden Unterklasse [Normschacht, Spezialbauwerk, Versickerungsanlage, Einleitstelle] wird hinzugefügt und verknüpft

  3. ein neuer Deckel-Bauwerksteil wird hinzugefügt und mit der Abwasserbauwerk verknüpft

  4. ein neuer Abwasserknoten wird in den Abwassernetzelementen und dessen Unterklasse Abwasserknoten erstellt

Wenn Sie zusätzliche Objekte wie Bauwerkteile ( Einstiegshilfe, Bankett, Rückstauklappen usw.) hinzufügen wollen, dann müssen sie zuerst den Datensatz sichern und danach das Abwasserbauwerk editieren (vgl. das Kapitel Bearbeiten von bestehenden Daten ).
