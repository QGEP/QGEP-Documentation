1.2.7. Digitalisierung der Einzugsgebiete¶

1.2.7.1. Allgemeines¶

Die Einzugsgebiete können mit den QGIS Standard Werkzeugen digitalisiert werden.

Einzugsgebiete dürfen sich nicht überlappen und können 4 Verknüpfungen zu Abwasserknoten haben.

1.2.7.2. Digitalisieren¶

  • Wähle den Enable Snapping Knopf

  • Öffne die Projekt Objektfang Einstellungen, schalte den Layer od_catchment_area und schalte die Schnittpunkte vermeiden Checkbox für den Layer od_catchment_area ein

Bemerkung

Neu erfasst Einzugsgebiete überlappen nicht mit bestehenden Einzugsgebieten dank diesen Einstellungen.

  • Setzen Sie den Layer od_catchment_area in den Bearbeitungsmodus

  • Benützen Sie das Objekt hinzufügen Werkzeug, um eine neue Fläche zu digitalisieren.

  • Start das Digitalisieren mit einem Links-Klick und fahre mit Links-Klick für jeden Stützpunkt des Polygons fort

  • Beende das Digitalisieren mit einem Rechts-Klick

Bemerkung

Bedenke, dass der Endpunkt des Polygon der letzte Punkt ist, wo ein Links-Klick verwendet wurde

  • Das od_catchment_area Objektattribute Fenster erscheint

  • Wenn Sie fertig sind klicken Sie den OK Knopf

Bemerkung

Das neue Einzugsgebiet überlappt nicht mit den bestehenden Einzugsgebieten.

Bemerkung

Um die Fläche eines ausgewählten Einzugsgebietes zu kontrollieren, kann im Abfrageergebnisse Fenster unter (abgeleitet) nachgeschaut werden.

Bemerkung

Für das Verknüpfen der Einzugsgebiete mit Abwasserknoten, vgl. das Kapitel Abwassernetzwerkselemente verbinden.

1.2.7.3. Bearbeitung¶

  • Sie können bestehende Bereiche mit dem Identify features Werkzeug bearbeiten.

Bemerkung

There are ideas to have a menu in the feature attribute window, that allows you to copy all current-values to the planned-fields. Not realized yet.

1.2.7.4. Einzugsgebiete teilen¶

  • Einzugsgebiete können mit dem Objekte zerteilen Werkzeug geteilt werden.

  • Das Einzugsgebiet muss nicht zuerst selektiert werden. Mit Links-Klicks wird die Trennlinie definiert. Mit einem Rechts-Klick wird abgeschlossen, wenn der letzte Punkt der Trennlinie abgesetzt ist.

1.2.7.5. Einzugsgebiet zusammenführen¶

Einzugsgebiet können mit dem Gewählte Objekte verschmelzen Tool zusammengeführt werden.

  • Wählen Sie od_catchment_area und wechseln Sie zum Bearbeitungsmodus

  • Mindestens zwei Einzugsgebiet auswählen, die zusammengeführt werden sollen

  • Click the Merge selected Feature tool. -> The Merge Feature Attributes windows appears.
  • Choose which values will be taken from which old record and click OK.

Achtung

Wenn einer der Datensätze gewählt wird und der Knopf Attribute des gewählten Objektes übernehmen geklickt wird, dann wird das Feld obj_id standardmässig übersprungen und die Datenbank wird die Änderung nicht akzeptieren. Wenn dieser Knopf benutzt wird, muss für das obj_id-Feld manuell ein Wert definiert werden.

1.2.7.6. Einzugsgebiete mit Abwasserknoten verbinden¶

Einzugsgebiete werden mit dem Tool Connect wastewater networkelements verknüpft (es ist das gleiche Tool mit dem Haltungen und Abwasserknoten verknüpft werden, vgl. den Beschrieb im Kapitel Bearbeiten in QGEP).

  • Es spielt keine Rolle, welcher Layer ausgewählt ist

  • Der Layer catchment_area muss im Bearbeitungsmodus sein

  • Klicke den Verbinde Abwassernetzwerkelement Knopf

  • Wenn der Mausezeiger über eine Einzugsgebiets-Grenze fährt, dann wird das Einzugsgebiet markiert: jetzt kann man klicken

  • Den Mausezeiger zu einem Abwassernetzelement führen, mit dem das Einzugsgebiet verknüpft werden soll (wieder werden wählbare Objekte markiert) und dort wieder klicken

  • Ein Fenster erscheint, in dem gewählt werden kann, welche Verknüpfung(en) definiert wird.

  • Wenn die Verknüpfung gespeichert wurde, erscheint ein blauer Mitteilungsbalken oben im Kartenfenster

Bemerkung

Im VSA-DSS-Datenmodell können Einzugsgebiete mit Abwasserknoten oder mit Haltungen verknüpft werden. Weil viele Hydraulikprogramme die Verknüpfung mit Haltungen nicht unterstützen, verknüpft dieses QGEP-Tool nur mit Abwasserknoten und nicht mit Haltungen.

Bemerkung

The tool does not check, if your choosen wastewater node is a part of the pwwf-network (primary network). It’s only the user who decides, if a node is ok or not. If you are not sure, please check the VSA Wegleitung GEP-Daten, part Erfassungsgrundsätze

Mit dem Tool können nicht mehrere Einzugsgebiete auf einmal mit einem Abwasserknoten verknüpft werden. Wenn das gewünscht ist, müssen die Einzugsgebiet ausgewählt werden, dann die Attributtabelle öffnen und den Mehrfach-Editier-Modus starten. Der gewünschte Abwasserknoten kann dann aus der Liste der vorhandenen Abwasserknoten gewählt werden

Um eine Verknüpfung mit einem Abwasserknoten zu löschen, kann das Einzugsgebiet mit dem Identify features Werkzeug bearbeiten werden.
