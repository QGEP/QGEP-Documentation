2.2.5. Kanäle Digitalisieren¶

2.2.5.1. Allgemeines¶

QGEP hat einen Assistenten, um Kanäle korrekt zu erfassen und diese mit Schächten respektive Abwasserknoten oder Haltungen zu verbinden (und dabei die Abwasserknoten- und Haltungs-Topologie aufzubauen). Siehe Kapitel wizard.

Wählen Sie die Wizard Schaltfläche, drücken Start Data Entry und wählen Reach im Pulldown-Menü.

Achtung

Erfassen Sie Haltungen/Kanäle immer in Fliessrichtung des Kanales, indem sie beim von Abwasserknoten starten und beim nach Abwasserknoten aufhören.

2.2.5.2. Digitalisieren¶

  • Im Digitalisierungsmodus schnappt der Cursor automatisch auf das nächstgelegene Abwasserbauwerk oder Haltung zu. Mit einem Linksklick wird ein Linienanfang gezeichnet.

  • Mit beliebigen weiteren Klicks können Sie Zwischenpunkte des Haltungsverlaufs definieren. Sie können auch direkt einen anderen Kanalschacht anwählen, um einen geraden Kanal zu zeichnen.

  • Die Advanced Digitizing Werkzeuge können zusammen mit dem Wizard eingesetzt werden.

  • Mit einem Rechtsklick wird die Linienzeichnung beendet. Im Anschluss erscheint das vw_qgep_reach Formular in dem Sie Daten im Reiter Allgemein eingeben können:

Bemerkung

Beachten Sie, dass der Endpunkt der Linie der Punkt ist, an welchem Sie zuletzt Links geklickt haben. Deshalb benötigen Sie, für die Digitalisierung einer einfachen Linie mit 2 Punkten, zwei Links Klicks um die Linie zu digitalisieren und einen Rechts Klick, um die Digitalisierung der Linie zu beenden.

  • Das Schnappen auf Abwasserknoten oder andere Haltungen kann in den Feldern HP_von/nach_FS_Abwassernetzelement kontrolliert werden.

Bemerkung

Wenn im Feld Bezeichnung (reach-identifier) im Reiter Allgemein keine Eingabe gemacht wird, übernimmt QGEP die obj_id als Bezeichnung (kann später geändert werden). Als Vorgabewert ist die Bezeichnung der Haltung gleich dem ws_identifier des Kanals. Die Bezeichnung der Haltungspunkte wird aus der Haltungsbezeichnung+‘F‘ oder +‘T‘ (von und nach Haltungspunkt) gebildet.

Bemerkung

Siehe Geometrieabgleichung für automatische Haltungspunkthöhen bei Schnappen auf Abwasserknoten mit Koten.

  • Für den Profiltyp erhalten Sie eine Liste von vordefinierten Profilen. Sie können diese in der od_pipe_profile Tabelle bearbeiten.

  • Wenn Sie fertig sind klicken Sie den OK Knopf

  • Speichern Sie die Informationen des Layers, indem Sie auf **Stop Data Entry ** klicken.

Bemerkung

Bei Verwendung des Assistenten während der Erstellung von Haltungen verwenden die Vorgabefelder im Reiter Allgemein (und ausschliesslich diese Felder) den zuletzt eingegebene Wert. Die Auswahlmöglichkeit Letzte Attributeingaben wiederverwenden* von QGIS hat hier keine Auswirkung.

  • Sie können einen Datensatz bearbeiten, indem Sie den Bearbeitungsmodus auswählen und dann mit dem Info Cursor auf das Objekt klicken, welches Sie bearbeiten möchten.

  • Wenn Sie den Bearbeitungsmodus nicht auswählen, können Sie nur die existierenden Daten anschauen.

  • Detaillierte Informationen zum Bearbeiten siehe das Kapitel Bearbeiten von bestehenden Daten.

2.2.5.3. Weitere Attribute und Klassen¶

Wenn eine Haltung digitalisiert wird, finden im Hintergrund eine Reihe von Schritten in der QGEP Datenbank statt:

  • Ein neues Objekt wird in der Klasse Abwasserbauwerk („od_wastewater_structure“) hinzugefügt

  • Ein neues Objekt wird hinzugefügt und verknüpft in der Unterklasse Kanal (od_channel)

  • Ein neues Haltungsobjekt wird in der Tabelle Abwassernetzelement (od_wastewater_networkelement) und dessen Unterklasse Haltung (od_reach) generiert.

  • Zwei neue Haltungspunktobjekte werden hinzugefügt und verknüpft mit der Haltung (rp_from_node, rp_to_node)

2.2.5.4. Geometrieabgleichung¶

Start- und Endpunkt der Linien-Geometrie bestimmen die Geometrie der Haltungspunkte. Die Höhe (Z Wert der Geometrie) des Start- und Endpunktes der hinzugefügten Linie wird über die Koten HP_von_Kote und HP_nach_Kote definiert. Das heisst, der Z Wert der Geometrie des Haltungspunktes wird ebenso über den Kote-Wert gesetzt. Diese Werte können über das „geschnappte“ Objekt gesetzt werden:

The snapped features 9405 and 9390 have the altitude from the wastewater node bottom_level. These values are copied into the attributes rp_from_level and rp_to_level and can be edited there by the user. When the user changes these values, the Z value of the geometry of the digitized feature and its reach points will be changed as well.

Bemerkung

Wenn die Kote eines Haltungpunktes ändert, ändert sich auch der Z Wert seiner Punkt-Geometrie und somit auch die Geometrie des Start- bzw. Endpunktes der Haltungslinie. Wenn der Z Wert der Geometrie der Haltung ändert, ändert auch die Geometrie und die Kote der Haltungspunkte. Wenn beide Werte gleichzeitig ändern, hat das Feld Kote Vorrang. Beim Einfügen ist es wie wenn beide Werte ändern. Also wird die Geometrie der Haltung gemäss der Koten der Haltungspunkte gesetzt und falls diese NULL sind, werden die Z Werte des Start- und Endpunktes der Haltung auf NaN gesetzt.
