2.2.6. Objekte löschen¶

Sie können Punkt- oder Linienobjekte (mit allen zugehörigen Objekten) mit den vw_ggep_(wastewater_structure/reach) Ebenen löschen. Sie können auch nur ein spezifisches Objekt auswählen und löschen (z.B. Deckel).

2.2.6.1. Haltungen löschen¶

  • Wählen Sie vw_qgep_reach und wechseln Sie zum Bearbeitungsmodus.

  • Wählen Sie die Haltung(en), die Sie löschen möchten. Sie können auf ein Objekt drücken oder eine Fläche zeichnen.

  • Benutzen Sie das Menü Bearbeiten –> Ausgewähltes löschen zum löschen.

Bemerkung

Am einfachsten ist es, die Delete Taste auf Ihrer Tastatur zu drücken. Oder Sie können den Delete* Knopf anklicken, welcher sich in der **Digitizing toolbar befindet.

Abhängig von ihren QGIS– Einstellungen kann die Digitizing Toolbar versteckt sein oder sich an einem anderen Ort befinden.

  • Deaktivieren Sie den Bearbeitungsmodus und bestätigen Sie die Änderungen an den Ebenen. Alle Änderungen werden in der Datenbank gespeichert.

2.2.6.2. Schächte und andere Abwasserbauwerke löschen¶

  • Wählen Sie vw_qgep_wastewater_structure und wechseln Sie zum Bearbeitungsmodus.

  • Wählen Sie das Objekt (manholes, special structures etc.), welches Sie löschen möchten

  • Zum löschen benutzen Sie Bearbeiten –>**Ausgewähltes löschen** .

  • Beenden Sie den Bearbeitungsmodus und bestätigen Sie die Änderungen an den Ebenen. Alle Änderungen werden in der Datenbank gespeichert.

2.2.6.3. Deckel löschen¶

  • Wählen Sie vw_qgep_wastewater_structure und wechseln Sie zum Bearbeitungsmodus.

  • Wählen Sie das Objekt (manholes, special structures etc.) von welchem Sie den/einen Deckel löschen möchten.

  • Benutzen Sie das Werkzeug Objekte abfragen, um das Formular zu öffnen

  • Wechseln Sie zum Deckel Tab

  • Wählen Sie den(die) Deckel, welche Sie löschen möchten

  • Klicken Sie auf den roten X Knopf, um die Deckel zu löschen

  • Klicken Sie im Formular auf Speichern

  • Deaktivieren Sie den Bearbeitungsmodus und bestätigen Sie die Änderungen an den Ebenen. Alle Änderungen werden in der Datenbank gespeichert.

2.2.6.4. Bauwerksteile löschen¶

  • Wählen Sie vw_qgep_wastewater_structure und wechseln Sie zum Bearbeitungsmodus.

  • Wählen Sie das Objekt (manholes, special structures etc.) von welchem Sie die Bauwerksteile löschen möchten.

  • Benutzen Sie das Werkzeug Objekte abfragen, um das Formular zu öffnen

  • Wechseln Sie zum Tab “Bauwerksteile”

  • Wählen Sie die Bauwerksteile, welche Sie löschen möchten

  • Klicken Sie auf das rote X um die Bauwerksteile zu löschen

  • Klicken Sie im Formular auf Speichern

  • Beenden Sie den Bearbeitungsmodus und bestätigen Sie die Änderungen an den Ebenen. Alle Änderungen werden in der Datenbank gespeichert.
