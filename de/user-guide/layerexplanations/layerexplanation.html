2.1.1. Layer Erläuterungen¶

  • vw_qgep_wastewater_structure (Abwasserbauwerk)

Main layer for covers of manholes, special structures, infiltration installations, discharge points (and wastewater treatment plant (wwtp) structure). Creating a new record in this layer always creates a new cover, a new wastewater node and a new wastewater structure. In the edit form, you have access to a lot of the related tables (e.g. structure parts, maintenance events).

  • vw_qgep_reach (Haltung)

Haupt-Layer für Linien-Abwasserbauwerke (Kanäle). Wird ein neuer Datensatz in diesem Layer erzeugt, wird eine neue Haltung und ein neuer Kanal erstellt. In der Eingabemaske ist Zugang zu vielen verknüpften Tabellen (z.B. Bauwerksteile, Erhaltungsereignisse).

  • Wastewater Structures / od_wastewater_structure (Abwasserbauwerk):

This layer shows the detailed geometries of wastewater structures. You digitize a new detailed geometry by the records of the layer vw_qgep_wastewater_structure. If you want to change an existing geometry, then you have to edit in the layer od_wastewater_structure.

  • Wastewater Structures / Structure Parts (Bauwerksteile):

Deckel ist der einzige Bauwerksteil mit einer eigenen Lageinformation. Alle anderen Bauwerksteile sind nur mit ihrem Abwasserbauwerk verknüpft und sollen nur über die Hauptlayer (vw_qgep_wastewater_structure und vw_qgep_reach) geändert werden.

  • Wastewater Structures / Structure Parts / vw_cover (Deckel)

Den Deckellayer nur benutzen, um die Lageinfo von einem ausgewählten Deckel zu verändern (und nicht das ganze Abwasserbauwerk zu verschieben). Oder um einen neuen Deckel zu einem Abwasserbauwerk hinzu zu fügen. (Ein Deckel kann auch im covers-Tab von vw_qgep_wastewater-structure hinzugefügt werden, aber dieser Deckel hat dann keine Lageinformation...).

  • Wastewater Structure / vw_channel (Kanal):

Wird dieser Layer für etwas benutzt oder kann er gelöscht werden?

  • Wastewater Structure / od_organisation (Organisation):

Diese Tabelle ist zur Zeit an einem komischen Ort und muss revidiert werden.

  • Inspection:

These tables are used in the maintenance tabs in the main tables. Do not edit directly.

  • Value Lists (Wertelisten):

Diese Wertelisten sind im Datenmodell VSA-DSS definiert. Nicht ändern.

  • Hydraulic / vw_wastewater_node (Abwasserknoten):

Den Abwasserknotenlayer nur benutzen, um die Lageinfo von einem ausgewählten Abwasserknoten zu verändern (und nicht das ganze Abwasserbauwerk zu verschieben). Oder um einen neuen Abwasserknoten zu einem Abwasserbauwerk hinzu zu fügen. (Ein Abwasserknoten kann auch im wastewater nodes-Tab von vw_qgep_wastewater-structure hinzugefügt werden, aber dieser Abwasserknoten hat dann keine Lageinformation...). Dieser Layer muss in Zukunft noch mit weiteren verknüpften Tabellen erweitert werden (z.B. Überlauf-Tabellen...)

  • Topology:

Diese beiden Layer werden von der qgep-Extension für das Längenporfil und die Netzverfolgung verwendet. Nicht verändern oder sonst benutzen.

  • Catchment (Einzugsgebiet):

Hauptlayer um Einzugsgebiete zu erfassen oder zu verändern.
