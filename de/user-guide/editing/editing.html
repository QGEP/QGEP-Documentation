2.2.1. Bearbeiten von bestehenden Daten¶

Eine Anleitung zum Bearbeiten von vorhandenen Daten in QGEP.

2.2.1.1. Demonstrationsprojekt¶

  • Gehen Sie sicher, dass sie das Demonstartionsprojekt mit pgAdminIII (siehe Initialisierung der Datenbank Kapitel) importiert haben

  • Öffnen sie das Demonstrationsprojekt indem Sie ins Hauptmenu gehen und dann Project –> Open drücken oder indem sie “STRG” + “O” gemeinsam drücken.

  • In der Datenbank anmelden

  • Sie sollten nun das Demonstrationsprojekt geladen haben

  • Das Projekt besteht aus verschiedenen Ebenen. Abhängig ihrer Aufgabe welches Sie bearbeiten oder hinzufügen wählen Sie die Ebene

2.2.1.2. Ebenen¶

  • QGEP hat verschiedene vordefinierte Ebenen:

2.2.1.2.1. vw_qgep_cover Ebene¶

  • Die vw_qgep_cover Ebene erlaubt es uns alle Punktelemente zu bearbeiten. Das sind Abwasserbauten wie zum Beispiel Kanalschächte, Sonderbauwerke, Entlastungspunkte und Versickerung Installationen (wwtp_structures, welches auch eine Abwasserbaute ist aber noch nicht unterstützt wird).

2.2.1.2.2. vw_qgep_reach Ebene¶

  • Die Ebene vw_qgep_reach ermöglichts uns alle linearen Elemente des Abwassernetzwerkes zu bearbeiten (**reaches / channels).

2.2.1.2.3. Abwasserbauten Ebene¶

  • Diese Ebene beinhaltet verschiedene Unterebenen in Verbundenheit mit der Abwasserbaute. Es beinhaltet auch Verbindungen zu den Unterhalt Veranstaltungen, die Verbindungstabelle zwischen Abwasserbauten und den Unterhalts Veranstaltungen und die Organsatio der Tabellen (wo owners, dataowners, etc gespeichert sind)

2.2.1.2.4. Werteliste Ebene¶

  • Hier können sie verschiedene Wertelisten sehen, welche mit den Klassen der Kanalisationsnetzwerken in Verbindung stehen.

2.2.1.2.5. Hydraulik Ebene¶

  • Diese Ebene erlaubt es uns Abwasserknoten auszuwählen und zu bearbeiten. Wählen Sie die Ebene und dann wählen Sie das Identify Features Werkzeug und dann Doppelklicken Sie auf deinen Knoten

2.2.1.2.6. Topologie Ebene¶

  • Hier finden Sie alle Informationen der Topologie des Netzwerkes

2.2.1.3. Attribute der Punktelemente ändern (Kanalschächte / Sondebauwerke)¶

  • Um einen Attribut zu ändern müssen Sie zuerst die vw_qgep_cover Ebende und wechseln Sie zum Bearbeitungsmodus.

  • Oder starten Sie den QGEP Wizard und klicken sie auf Start Data Entry

  • Dann klicken Sie auf den Identify feautures Knopf und wählend Sie das Element, welches Sie bearbeiten wollen, mit einem Doppelklick

  • Das vw_qgep_cover Formular öffnet sich.

  • Auf dem ersten Tab können Sie üblichsten attribute ändern

  • Um die Attribute von in Verbindung stehenden Tabellen (z.B Abdeckungen) zu ändern, müssen Sie den entsprechenden Tab auswählen. Abhängig von der Art der Abwasserbaute können Sie zusätzliche Attribute im zugehörigen Tab bearbeiten (special structure, manhole).

  • Der covers Tab erlaubt ihnen zusätzliche Abdeckungen hinzuzufügen.

  • Im structure parts* können Sie auch weitere Teile wie zum Beispiel Access aids, backflow prevention, dryweather flume etc. hinzufügen

  • Im **wastewater nodes part* können Sie alle Attribute dieser Klasse bearbeiten oder hinzufügen oder ein zweites Element hinzufügen.

2.2.1.4. Attribute der Linienelemente ändern (Kanäle)¶

  • Um einen Attribut zu ändern müssen Sie zuerst die vw_qgep_cover Ebende und wechseln Sie zum Bearbeitungsmodus.

  • Dann klicken Sie auf den Identify feautures Knopf und wählend Sie das Element, welches Sie bearbeiten wollen, mit einem Doppelklick auf einen Kanal.

  • Das vw_qgep_reach Formular öffnet sich.

  • Auf dem ersten Tab können Sie üblichsten Attribute ändern.

  • Um die Attribute von in Verbindung stehenden Tabellen (z.B reach points) zu ändern, wählen Sie den entsprechenden Tab.

2.2.1.5. Attribute der Hydraulikelemente ändern¶

  • Aktiviert mit dem Identify feature Werkzeug, werden durch Rechtklicken eine Auswahl aller existierender Objekte ausgegeben.

  • Sie können dann das Objekt, welches Sie wollen auswählen. Das bringt Sie zum zugehörigen Formular um die Details einzusehen

2.2.1.6. Änderungen Speichern¶

  • Drücken Sie Save und dann deaktivieren Sie den Bearbeitungsmodus oder klicken Sie auf Stop data entry wenn Sie mit dem QGEP Data Entry Arbeitsassistenten gearbeitet haben.

..* Vorgehen:

  • Changing relations (Verknüpfungen ändern)

  • Ändern ws_type (Subklasse eines Abwasserbauwerkes ändern (ObjektID ändert dann auch!))

  • Kanäle in verschiedene Abschnitte unterteilen (Kanäle unterteilen)
