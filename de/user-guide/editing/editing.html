2.3.1. Bearbeiten von bestehenden Daten¶

Eine Anleitung zum Bearbeiten von vorhandenen Daten in QGEP.

2.3.1.1. Demoprojekt¶

  • Gehen Sie sicher, dass sie das Demoprojekt mit pgAdminIII (siehe Initialisierung der Datenbank Kapitel) importiert haben

  • Öffnen sie das Demoprojekt indem Sie ins Hauptmenu gehen und dann Project –> Open drücken oder indem sie “STRG” + “O” gemeinsam drücken.

  • In der Datenbank anmelden

  • Sie sollten nun das Demonstrationsprojekt geladen haben

  • Das Projekt besteht aus verschiedenen Ebenen. Abhängig ihrer Aufgabe welches Sie bearbeiten oder hinzufügen wählen Sie die Ebene

2.3.1.2. Ebenen¶

  • QGEP hat verschiedene vordefinierte Ebenen:

2.3.1.2.1. vw_qgep_cover Ebene¶

  • Die vw_qgep_cover Ebene erlaubt es uns alle Punktelemente zu bearbeiten. Das sind Abwasserbauten wie zum Beispiel Kanalschächte, Sonderbauwerke, Entlastungspunkte und Versickerung Installationen (wwtp_structures ist auch eine Abwasserbauwerk, wird aber noch nicht unterstützt ).

2.3.1.2.2. vw_qgep_reach Ebene¶

  • Die Ebene vw_qgep_reach ermöglicht es uns, alle linearen Elemente des Abwassernetz zu bearbeiten (**reaches / channels).

2.3.1.2.3. Ebene Abwasserbauwerke¶

  • Diese Ebene beinhaltet verschiedene Unterebenen die mit Abwasserbauwerken verknüpft sind. Es beinhaltet auch Verbindungen zu den Erhaltungsereignissen, die Verbindungstabelle zwischen Abwasserbauten und den Erhaltungsereignissen und der Tabelle Organsation (wo owners, dataowners, etc gespeichert sind)

2.3.1.2.4. Ebene Werteliste¶

  • Hier können sie verschiedene Wertelisten sehen, welche mit den Klassen der Kanalisationsnetzwerken in Verbindung stehen.

2.3.1.2.5. Hydraulik Ebene¶

  • Diese Ebene erlaubt es, Abwasserknoten auszuwählen und zu bearbeiten. Wählen Sie die Ebene und dann wählen Sie das Identify Features Werkzeug und dann Doppelklicken Sie auf den Knoten

2.3.1.2.6. Topologie Ebene¶

  • Hier finden Sie alle Informationen der Topologie des Netzwerkes

2.3.1.3. Attribute der Punktelemente ändern (Normschächte / Spezialbauwerke)¶

  • Um einen Attribut zu ändern müssen Sie zuerst die vw_qgep_cover Ebene auswählen und wechseln Sie zum Bearbeitungsmodus.

  • Oder starten Sie den QGEP Wizard und klicken sie auf Start Data Entry

  • Dann klicken Sie auf den Identify features Knopf und wählen das Element, welches Sie bearbeiten wollen, mit einem Doppelklick

  • Das vw_qgep_cover Formular öffnet sich.

  • Auf dem ersten Tab können Sie die am häufigsten benutzten Attribute ändern

  • Um die Attribute von in Verbindung stehenden Tabellen (z.B Deckel) zu ändern, müssen Sie den entsprechenden Tab auswählen. Abhängig von der Art des Abwasserbauwerks können Sie zusätzliche Attribute im zugehörigen Tab bearbeiten (special structure, manhole).

  • Der covers Tab erlaubt ihnen zusätzliche Deckel hinzuzufügen.

Es ist nicht empfohlen, an dieser Stelle einen weiteren Deckel hinzuzufügen, weil die Lage des neuen Deckels nicht definiert werden kann.

  • Unter Bauwerkteile/structure parts können Sie weitere Bauwerksteile wie zum Beispiel Einstiegshilfen/access aids, Rückstausicherung/backflow prevention, Trockenwetterrinne/dryweather flume etc. hinzufügen

  • Unter **Abwasserknoten/wastewater nodes part* können Sie alle Attribute dieser Klasse bearbeiten oder hinzufügen oder einen weiteren Abwasserknoten hinzufügen.

Es ist nicht empfohlen, an dieser Stelle einen weiteren Abwasserknoten hinzuzufügen, weil die Lage des neuen Abwasserknotens nicht definiert werden kann.

2.3.1.4. Attribute der Linienelemente ändern (Kanäle)¶

  • Um einen Attribut zu ändern müssen Sie zuerst die vw_qgep_cover Ebene auswählen und wechseln Sie zum Bearbeitungsmodus.

  • Dann klicken Sie auf den Identify feautures Knopf und wählen das Element, welches Sie bearbeiten wollen, mit einem Doppelklick auf einen Kanal.

  • Das vw_qgep_reach Formular öffnet sich.

  • Auf dem ersten Tab können Sie am häufigsten benutzte Attribute ändern.

  • Um die Attribute von in Verbindung stehenden Tabellen (z.B Haltungspunkte/reach points) zu ändern, wählen Sie den entsprechenden Tab.

2.3.1.5. Attribute der Hydraulikelemente ändern¶

  • Aktiviert mit dem Identify feature Werkzeug, wird durch Rechtsklicken eine Auswahl aller existierender Objekte ausgegeben.

  • Sie können dann das gewünschte Objekt auswählen. Es erscheint das zugehörige Formular um die Details einzusehen

2.3.1.6. Hinzufügen von weiteren Deckel- oder Abwasserknoten-Datensätzen zu bestehenden Abwasserbauwerken¶

Dieser Beschrieb ist für einen zusätzlichen Deckel.

  • Voraussetzung: Unter Projekt/Projekteigenschaften/Beziehungen muss die Beziehung zwischen vw_cover (Feld fk_wastewater_structure) und vw_qgep_wastewater_structure (Feld obj_id) vorhanden sein.

  • Im Layer vw_cover / Eigenschaften /Felder muss das Feld fk_wastewater_structure als Beziehungsreferenz definiert sein.

  • Die Bearbeitung des Layer vw_cover starten und einen neuen Deckel definieren.

  • In der Form hat es hinter dem Feld fk_wastewater_structure einen Knopf, mit dem der neue Deckel mit einem bestehenden Abwasserbauwerk verknüpft warden kann.

2.3.1.7. Änderungen Speichern¶

  • Drücken Sie Save und dann deaktivieren Sie den Bearbeitungsmodus oder klicken Sie auf Stop data entry wenn Sie mit dem QGEP Data Entry Arbeitsassistenten gearbeitet haben.
