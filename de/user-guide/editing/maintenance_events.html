2.3.3. Hinzufügen oder bearbeiten von Erhaltungsereignissen¶

Eine Anleitung zum hinzufügen oder bearbeiten vom Erhaltungsereignissen Datensätzen in QGEP.

2.3.3.1. Allgemeines¶

  • Erhaltungsereignissen können mit einem oder mehreren Abwasserbauwerken verbunden sein. In der Datenbank ist das mit einer re-Zwischentabelle umgesetzt, um die n:m-Beziehung abzubilden (re_maintenance_event_wastewater_structure)

2.3.3.2. Hinzufügen ( Für Einzeltermine welche nur zu einem Bauwerk gehören)¶

  • Aktivieren Sie den Bearbeitungsmodus.

  • Mit dem Identify Features Werkzeug können Sie manholes oder reaches auswählen indem sie ihre zugehörige Ebene auswählen (**vw_qgep_cover oder vw_qgep_reach)

  • Das entsprechende Formular öffnet sich. Wechseln Sie zum Erhaltungsereignisse -Reiter

  • Erstellen Sie ein neues Erhaltungsereignis.

  • Oder verknüpfen Sie es mit einem bereits bestehenden Erhaltungsereignis.

  • Schliessen Sie das Formular mit OK und beeden Sie den Bearbeitungsmodus um die Änderungen zu speichern

2.3.3.3. Hinzufügen ( für mehrere Wartungsereignisse, welche zu mehreren Bauwerken gehören können)¶

  • Vorgehen

2.3.3.4. Bearbeitung¶

  • Machen Sie es wie es beim Hinzufügen beschrieben - aber bearbeiten Sie nur die bereits bestehenden Erhaltungsereignisse
