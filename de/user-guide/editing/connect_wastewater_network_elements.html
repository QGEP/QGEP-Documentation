2.3.4. Abwassernetzwerkselemente verbinden¶

Eine Anleitung zur Verbindung von Abwassernetzelementen in QGEP

2.3.4.1. Allgemeines¶

  • QGEP verfügt über ein Werkzeug um Abwassernetzelemente zu verbinden

2.3.4.2. Ein Haltung mit einem Knoten oder einer anderen Haltung verbinden¶

  • Es spielt keine Rolle, welcher Layer ausgewählt ist

  • Der Layer vw_qgep_reach muss im Bearbeitungsmodus sein

  • Klicke den Verbinde Abwassernetzwerkelement Knopf

  • Wenn der Mausezeiger über ein Haltung fährt, die verknüpft werden kann, dann wird die Haltung markiert: jetzt kann man klicken

  • Den Mausezeiger zu einem Abwassernetzelement führen, mit dem das erste Objekt verknüpft werden soll (wieder werden wählbare Objekte markiert) und dort wieder klicken

  • Ein Fenster erscheint, in dem gewählt werden kann, welche Verknüpfung definiert wird. Wenn eine Haltung verknüpft wird, sieht das Fenster so aus:

  • Wenn die Verknüpfung gespeichert wurde, erscheint ein blauer Mitteilungsbalken oben im Kartenfenster

2.3.4.4. Eine Verknüpfung löschen¶

Zur Zeit gibt es kein spezielles Werkzeug, um eine Verknüpfung zwischen Abwassernetzwerkelementen zu löschen.

  • Wähle den Layer vw_qgep_reach und aktiviere den Bearbeitungsmodus

  • Wähle das Info Werkzeug und doppelklicke die Haltung, welche geändert werden soll

  • Lösche den Wert in Feld rp_..._fk_wastewater_networkelement, um die Verknüpfung zu löschen

Bemerkung

Beachte die ‘VSA Erfassungsgrundsätze’: Ein von-Haltungspunkt (rp_from_fk_wastewater_networkelement) soll nie mit einer anderen Haltung verknüpft werden.

Beispiel: Wenn eine Haltung direkt mit einer anderen Haltung verknüpft ist ohne ein Schachtbauwerk, dann soll die obere Haltung mit der unteren Haltung verknüpft werden (rp_from_fk_wastewater_networkelement = downstream reach). Aber die untere Haltung darf in diesem Fall nicht mit der oberen Haltung verknüpft werden. Diese Regel gilt für SAA-Kanäle (sekundäres Netzwerk). In einem PAA-Netzwerk sind immer Knoten zwischen den Haltungen verknüpft.
