5.1. Über

Eine Anleitung zum Herunterladen und Installieren einer virtuellen Maschine (VM) welche eine funktionierende QGEP Installation mit einem Demo Datensatz besitzt. Die VM besitzt auch eine funktionierende Demo des QWAT water project .

Die VM basiert auf Debian Test mit dem ‚Cinnamon‘ Desktop.

Debian Testing wurde ausgesucht um über GDAL 2.x zu verfügen und auch vom aktuellsten Linux Betriebssystem zu profitieren, damit die aktuellste Hardware kein Problem darstellt, für den Fall dass die VM ausserhalb der Virtualisierung verwendet wird (z.B Benützung der Virtuellen Diskette als Basis zur Erstellung eines nicht virtualisierten USB-Stick welcher für Demonstrationzwecke verwendet wird).

Benötigte Software:

  • PostgreSQL 9.6.0 mit Postgis 2.3

  • QGIS 2.18

  • pgAdmin 4 (zum Zugriff ist der Webbrowser zu öffnen über http://localhost:5050)

Einstellungen welche bei einer Standard Softwareinstallation geändert werden:

  • pg_hba.conf verfügt über trust für lokale Verbindungen

  • QGIS ist Aufgebaut mit multithreaded rendering enabled

Bemerkung

Der Benutzer qgis und root besitzen das Password „gqis

Bemerkung

Wenn Sie isch entscheiden die VM in der produktion zu verwenden, dann ist es Empfohlen für:

  • ändern Sie die Passwörter

  • Regenerierung der ssh Tasten indem Sie es im Terminal sudo rm /etc/ssh/ssh_host_* && sudo dpkg-reconfigure openssh-server laufen lassen
