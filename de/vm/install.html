4.2. Installation¶

4.2.1. Herunterladen¶

Download the vm from here

Bemerkung

The size of the VM is around 2GB compressed and up to 15GB uncompressed.

4.2.2. Entpacken¶

Die virtuelle Diskette wird mit der Benutzung von XZ Komprimierung erzielt

Um das Archiv zu entpacken

  • Windows Sie können 7-Zip benutzen um das Archiv zu entpacken.

  • Linux: CD zum Downloadpfad und lassen Sie xz -d qgis.vdi.xz laufen

SIe können die virtuelle Maschine laufen lassen wie Sie es mit einer VirtualBox VW tun würden.

Bemerkung

Da die virtuelle Maschine auf Linux basiert ist es relativ einfach ein Bild in eine rohe Diskette zu konvertieren und Sie dann auf einen USB-Stick (Minimum 16GB Speicher) zu packen. Dies erlaubt es ihnen die App ohne den Virtualisations Nachteil auszuführen und die volle Geschwindigkeit ihrer Hardware zu geniessen.

4.2.3. QGEP ausführen¶

Nachdem Sie sich als “qgis” Benutzer und dem selben Passwort angemeldet haben, erhalten Sie durch Klicken auf das QGIS Symbol im unteren Balken folgendes Bild:

Das Klicken auf das qgp_en Projekt und das befolgen des QGEP Benutzerhandbuch sollte Sie ins laufen bringen.
