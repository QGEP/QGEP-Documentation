5.1. Acerca

Esto representa una guía como descargar e instalar una máquina virtual (VM) que posee una instalación operativa de QGEP con un conjunto de datos de demostración. La VM también posee una demostración operativa de QWAT water project .

La VM está basada en Debian Testing con Cinnamon Desktop.

Debian Testing se eligió para tener GDAL 2.x y también beneficiarse del último kernel de Linux para que el hardware más moderno no sea un problema en el caso de que la VM se use fuera de la virtualización (por ejemplo, usando el disco virtual como base para crear un usb-stick no virtualizado para ser utilizado con fines de demostración).

Software instalado:

  • PostgreSQL 9.6.0 con Postgis 2.3

  • QGIS 2.18

  • pgAdmin 4 (Abre el navegador en http://localhost:5050 para acceder al mismo)

Configuraciones que se cambiaron desde una instalación de software predeterminada:

  • pg_hba.conf posee trust para las conexiones locales

  • QGIS está configurado con multithreaded rendering enabled

Nota

Los usuarios qgis y root poseen la contraseña qgis

Nota

Si decide usar VM en producción, es recomendado usar:

  • cambiar las contraseñas

  • regenera las claves ssh ejecutando en la terminal sudo rm /etc/ssh/ssh_host_* && sudo dpkg-reconfigure openssh-server
