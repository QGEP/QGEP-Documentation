Servidor QGIS

Puedes descargar una VM con una versión añeja de QGEP que también posee un servidor QGIS instalado desde http://docs.qtibia.ro/qgisplatform.vdi.xz .

La VM cuenta con instalaciones operativas de Servidor QGIS y Cliente Web QGIS (QWC).

Acceso a los servicios Web

El servidor Apache está configurado para responder a las solicitudes que apuntan a http://qgisplatform.demo. El acceso al enlace anterior con el navegador de Internet de la máquina virtual te llevará a la página de inicio de QWC que se muestra en la imagen de abajo.

Si deseas acceder a los servicios web fuera de la máquina virtual (VM), necesitas modificar el archivo hosts en tu máquina y apuntar qgisplatform.demo a la IP de tu VM.

Nota

Si has configurado la VM con el adaptador de red en modo NAT, solo el host de la máquina virtual podrá acceder a los servicios. Si deseas que todos los dispositivos de tu LAN lo hagan, debes configurar el adaptador de red en modo puente.
