Inițializare bază de date¶

You can use pgAdmin to access and manage the database server.

Note

By clicking the link you can learn how to Install pgAdmin

Procedură¶

În pgAdminIII

  • Conectează-te la baza de date.

  • Create a new group role (preferably named qgep, but you are free to choose)
  • Create a new login role (qgepuser for example) and make it a member of the qgep group

You can do this graphically or by simply opening a pgAdminIII SQL query window and type the following:

CREATE GROUP qgep;
CREATE ROLE qgepuser LOGIN;
GRANT qgep TO qgepuser;
  • Create a new database (preferably named qgep, but you are free to choose)

    • Conectează-te la această bază de date

  • Creează o nouă schemă cu numele qgep

  • Open an SQL query Window and create the extensions (if they’re not created yet)
    • CREATE EXTENSION hstore;
    • CREATE EXTENSION postgis;

  • Descarcă datele de demo

    • http://qgis.vitu.ch/qgep_demodata/qgep.backup

  • Click dreapta pe schema qgep

    • Click pe restore

    • Încarcă fișierul qgep.backup

    • Restore Options #2: Activează Clean before restore

    • Click Restore
    • Click Cancel

  • Click dreapta pe baza de date urmat de click pe Refresh

  • Actualizează privilegiile bazei de date

    • Click dreapta pe schema qgep

    • Click Grant Wizard …

    • Selecție, Click Check All

    • Privilegii

      • Grupul qgep

      • Alege ALL
