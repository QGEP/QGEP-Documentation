6.7. Mapping¶

Use processing tool join attributes to add the results stored in Node summary and Link summary to the tables qgep_od.vw_wastewater_node and qgep_od.vw_qgep_reach. The mapping should be done on the attribute obj_id of the qgep_od tables and the attribute id of the summary tables.
