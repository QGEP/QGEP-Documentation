Install¶

Download¶

Download the vm from here

Note

The size of the VM is around 2GB compressed and around 7GB uncompressed.

Extract¶

The virtual disk is archived using XZ compression.

To extract the archive:

  • Windows: you can use 7-Zip to extract the archive.
  • Linux: cd to the download directory and run xz -d qgisplatform.vdi.xz

You can run the VM as you would do with any VirtualBox VM.

As the VM has Linux under it it is possible to also put the app on a USB stick (minimum 8GB) or a hard disk.
