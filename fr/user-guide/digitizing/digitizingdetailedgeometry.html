2.2.3. Numérisation de géométries détaillées¶

2.2.3.1. Général¶

Il est possible d’ajouter une forme géométrique à une structure assainissement.

Ceci est recommandé pour:

  • Les structures spéciales et les points d’infiltration, parce qu’il n’existe pas de structure normée comme pour les regards, pour lesquels vous pouvez définir les dimension1 et dimension2 pour représenter leur étendue.

  • Les installations d’infiltration pour les même raisons

Attention

Ce type de géométrie n’est généralement pas utilisé pour les regards de visite ou les canalisations!

2.2.3.2. Outil de numérisation¶

  • Sélectionnez la couche vw_qgep_wastewater_structure

  • Sélectionnez avec le curseur d’information l’objet auquel vous voulez rajouter une géométrie détaillée

  • Effectuez un clic droit sur l’objet

  • S’il y a plusieurs objets: sélectionnez la couche vw_qgep_wastewater_structure

  • Choisissez l’option DIgitize

  • Vous pouvez maintenant dessiner le polygone.

  • Faites un clic droit pour arrêter la numérisation.

Une autre méthode consiste à ouvrir le formulaire vw_qgep_wastewater_structure avec l’outil Info, puis de choisir Numériser dans le menu Actions du formulaire.
