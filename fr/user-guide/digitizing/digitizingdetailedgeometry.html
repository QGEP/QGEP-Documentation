2.1.3. Numérisation de géométrie élaborée¶

2.1.3.1. Général¶

Il est possible d’ajouter une forme géométrique à une structure.

Ceci est recommandé pour:

  • Les structures spéciales et les points d’infiltration, parce qu’il n’existe pas de structure normées comme pour les regards de visite pour les quels il suffit de définir les dimension1 et dimension2 pour représenter leur étendu.

  • Les installations d’infiltration pour les même raisons

Attention

Ce type de géométrie n’est généralement pas utilisé pour les regards de visite ou les canalisations!

2.1.3.2. Outils de numérisation¶

  • Select the vw_qgep_wastewater_structure layer

  • Sélectionnez avec le curseur d’information l’objet auquel vous voulez rajouter une géométrie détaillée

  • Right click the object
  • if there are several objects: select the vw_qgep_wastewater_structure layer

  • Choisissez l’option DIgitize

  • Vous pouvez maintenant dessiner le polygone.

  • Faire un clique droit pour arrêter la numérisation.

An other way is to open the vw_qgep_wastewater_structure form with the info tool and then choose Digitize in the Aktionen menu of the form.
