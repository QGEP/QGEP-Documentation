2.2.1. Saisie de données¶

Elle peut se faire de deux façons:

  • En construisant ou en numérisant les données dans le projet SIG QGEP directement

  • En saisissant les données dans un programme/logiciel externe puis en important les positions et/ou données techniques dans le modèle QGEP, par exemple:

    • coordonnées des chambres provenant de mesures de terrain

    • positions et/ou données techniques provenant d’un autre système SIG ou DAO

    • En utilisant les données de la mensuration (p. ex. “limites de parcelles”) comme base pour définir les limites des bassins versants.

    • autres

Bien qu’elle demande une certaine compréhension du modèle de données VSA-SDEE, la saisie de données dans QGEP est prise en charge par des outils d’assistance et des formulaires qui permettent de relier correctement les différentes tables. La relation entre les conduites et les structures est ainsi automatisée et peut être contrôlée à travers l’outil de poursuite du réseau.

2.2.1.1. Généralités¶

Enable the Reuse last entered attribute values option in Settings ‣ Options ‣ Digitizing.

Note

Lors de la numérisation, commencez par les éléments ponctuels (structures du réseau assainissement telles que regards, structures spéciales). Il sera ensuite facile de connecter ces éléments ponctuels avec des éléments linéaires (conduites avec tronçons).

Pour commencer la saisie de données, sélectionnez le bouton Wizard (assistant) de la barre d’outils QGEP

Dans le coin inférieur gauche, la fenêtre Saisie de données QGEP apparaît:

Cliquez sur Commencer la saisie de données pour basculer en mode d’édition.
