2.2.5. Numérisation des Canalisations¶

2.2.5.1. Général¶

Désactivez la case Réutiliser la dernière valeur attributaire saisie (Préférences -> Options -> Numérisation -> Création d’entités), pour éviter que la saisie de votre deuxième objet soit bloquée. Ce bug sera corrigé au plus tard dans QGIS 3.0.

QGEP propose un outil pour construire les canalisations et les connecter aux chambres/structures spéciales (tout en créant la topologie pour les nœuds de réseau et les conduites).

Attention

Commencer la digitalisation dans le sens de l’écoulement en commençant avec le from manhole node et en finissant par le to manhole node.

Note

Pour l’instant il n’existe pas d’outils permettant d’inverser le sens d’écoulement d’une conduite (topologiquement parlant), il faut donc faire attention à la construire correctement dès le début.

2.2.5.2. Numérisation¶

  • Une fois le mode de numérisation activé, le curseur va automatiquement s’accrocher à la structure ou conduite la plus proche. Cliquez sur le bouton gauche de la souris pour commencer à dessiner.

  • Utiliser le bouton gauche de la souris pour définir les points intermédiaires de la conduite. Vous pouvez aussi directement sélectionner une autre chambre pour dessiner une conduite droite.

  • Faire un clic droit pour terminer la numérisation. Ceci va faire apparaître le formulaire vw_qep_reach.

Note

Gardez en tête que le point final de la conduite est celui ou vous avez effectué le dernier clique gauche avec la souris, le clique droit servant seulement à terminer l’édition et faire apparaître le formulaire. Ainsi pour une conduite de 2 points, il faudra deux cliques gauches et un clique droit.

  • Remplissez les attributs dans l’onglet General. Pour le type de profils, vous pourrez choisir parmi une liste qu’il faut éditer dans la table od_pipe_profile.

  • Une fois que vous avez terminé, cliquez sur le bouton OK.

  • Sauvez les informations de la couche en stoppant l’outil d’assistance (“the wizard”).

  • Vous pouvez éditer à nouveau un objet en sélectionnant les boutons Basculer en mode édition et Identifier les entités avant de sélectionner l’objet désiré avec le curseur.

  • Si le mode d’édition n’est pas activé, vous pourrez seulement consulter les informations relatives à l’objet.

  • Pour de plus d’informations à propos de l’édition de données, voir le chapitre Modification de données existantes

2.2.5.3. Autres classes et attributs¶

Lorsqu’un objet de type ligne est créé, une série d’étapes prend place en arrière-plan de la base de données QGIS:

  • Un nouvel objet est ajouté dans la classe structure_assainissement (od_wastewater_structure)

  • Un nouvel objet est ajouté et lié dans la sous-classe channel (od_channel)

  • Deux objets de type reach point sont ajoutés et liés à la conduite (rp_from_node, rp_to_node)

  • Un nouvel objet tronçon est généré dans la classe wastewater network elements (“od_wastewater_networkelement”) / éléments du réseau assainissement (“od_élément_réseau_assainissement”) et sa sous-classe reach (“od_reach”) / tronçon (**od_tronçon”)
