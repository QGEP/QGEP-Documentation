2.2.6. Suppression d’objets¶

Vous pouvez supprimer des points ou des lignes (avec leurs objets associés) à l’aide de la couche vw_qgep_(wastewater_structure/reach). Vous pouvez aussi sélectionner et supprimer uniquement un objet spécifique (ex. couvercle).

2.2.6.1. Suppressions de conduites¶

  • Sélectionnez la couche vw_qgep_reach et passez en mode édition.

  • Sélectionnez la/les conduite/s que vous voulez supprimer. Vous pouvez cliquer sur l’objet ou dessiner une aire de sélection.

  • Utilisez le menu édition –> Supprimer les entités pour supprimer.

Note

La façon la plus simple est de presser la touche Delete du clavier. Une autre possibilité est de presser sur le bouton Supprimer qui se trouve dans la barre d’outils de digitalisation.

Selon votre configuration de la barre d’outils QGIS, il est possible que les outils de digitalisation ne se trouve pas au même endroit que sur l’image.

  • Stoppez le mode d’édition et confirmez les modifications apportées à la couche. Toutes les modifications vont alors être sauvegardées dans la base de données.

2.2.6.2. Suppression des chambres et autres structures¶

  • Sélectionner vw_qgep_wastewater_structure et passer en mode édition

  • Sélectionnez les objets (chambres, structures spéciales etc.) que vous voulez supprimer

  • Utilisez le menu édition –> Supprimer les entités pour supprimer.

  • Stoppez le mode d’édition et confirmez les modifications apportées à la couche. Toutes les modifications vont alors être sauvegardées dans la base de données.

2.2.6.3. Suppressions des couvercles¶

  • Sélectionner vw_qgep_wastewater_structure et passer en mode édition

  • Sélectionnez les objets (chambres, structures spéciales etc.) dont vous voulez supprimer le/les couvercles

  • Utilisez l’outil Sélectionner une entité pour ouvrir le formulaire

  • Allez dans l’onglet covers

  • Sélectionnez le/les couvercle/s que vous voulez supprimer

  • Cliquez sur le bouton x rouge pour supprimer

  • Cliquez sur le bouton sauvegarder du formulaire

  • Stoppez le mode d’édition et confirmez les modifications apportées à la couche. Toutes les modifications vont alors être sauvegardées dans la base de données.

2.2.6.4. Suppression des éléments de structures¶

  • Sélectionner vw_qgep_wastewater_structure et passer en mode édition

  • Sélectionnez les objets (chambres, structures spéciales, etc.) dont vous voulez supprimer un élément structurel

  • Utilisez l’outil Sélectionner une entité pour ouvrir le formulaire

  • Allez dans l’onglet structure parts

  • Sélectionnez le/les élément/s que vous voulez supprimer

  • Cliquez sur le bouton x rouge pour supprimer

  • Cliquez sur le bouton sauvegarder du formulaire

  • Stoppez le mode d’édition et confirmez les modifications apportées à la couche. Toutes les modifications vont alors être sauvegardées dans la base de données.
