2.2.7. Numérisation des bassins versants¶

2.2.7.1. Généralités¶

Les bassins versants peuvent être numérisés avec les outils standards de QGIS.

Les bassins versants ne doivent pas se chevaucher et peuvent avoir 4 connexions avec des noeuds assainissement.

2.2.7.2. Numérisation¶

  • Sélectionnez le bouton Activer l’accrochage

  • Ouvrez Réglages des options d’accrochage du projet, contrôlez la couche od_zone_bassin-versant et contrôlez la boîte à cocher éviter intersection dans cette même couche

Note

Les périmètres de bassins versants nouvellement numérisés ne vont pas chevaucher de périmètres existants avec ce réglage.

  • Passez la couche od_aire_bassin_versant en mode d’édition

  • Utilisez l’outil Ajouter une entité pour numériser un bassin versant.

  • Commencer la numérisation avec un clic gauche et continuer avec un clic gauche pour chaque sommet du polygone

  • Stoppez la numérisation avec un clic droit

Note

Gardez en mémoire que le point final du polygone est le dernier point où vous avez effectué un clic gauche.

  • Le formulaire od_zone_bassin-versant apparait

  • Lorsque vous avez terminé l’édition des attributs, cliquez sur le bouton OK

Note

La nouvelle zone ne chevauche pas les anciennes zones.

Note

Pour contrôler la zone exacte du polygone sélectionné, regardez dans la fenêtre (Dérivé) –> **Identifier résultats* sous **.

Note

Pour connecter la zone avec les noeuds assainissement, voir le chapitre connecter-éléments-réseau-assainissement.

2.2.7.3. Édition¶

  • Vous pouvez éditer des bassins existants avec l’outil Sélectionner une entité.

Note

Des réflexions sont en cours afin d’intégrer un menu (dans la fenêtre attribut des entités) qui permettrait de recopier toutes les valeurs des champs “courant” dans les champs “planifié”. Option non réalisée pour le moment.

2.2.7.4. Séparer des zones¶

  • Vous pouvez séparer des zones existantes avec l’outil Séparer.

  • Il n’est pas nécessaire de sélectionner la zone en premier. Effectuez un clic gauche pour définir la ligne de séparation. Effectuez un clic droit après avoir défini le dernier point de la ligne de séparation.

2.2.7.5. Fusionner des zones¶

Vous pouvez fusionner des zones existantes avec l’outil Fusionner objets sélectionnés.

  • Sélectionnez la couche od_zone_bassin-versant et basculez dans le mode d’édition

  • Sélectionnez au moins deux zones que vous voulez fusionner

  • Cliquez sur l’outil « Fusionner les entités sélectionnés ». La fenêtre Fusionner attributs entités apparaît.

  • Choisir quelles valeurs seront reprise à partir de quelle entrée existante et cliquer OK.

Attention

Si vous sélectionnez l’une des entrées et cliquez sur le bouton Reprendre les attributs de l’objet sélectionné, le champ id_obj sera ignoré par défaut et la base de données n’acceptera pas les modifications. Après avoir utilisé ce bouton, vous devez choisir manuellement l’une des valeurs du champ id_obj.

2.2.7.6. Connecter la zone du bassin versant aux noeuds du réseau assainissement¶

Vous connectez les bassins versants avec l’outil Connecter éléments réseau assainissement (c’est le même outil qui est utilisé pour connecter les tronçons et les noeuds, décrit dans le chapitre « Editer dans QGEP »).

  • Peu importe la couche sélectionnée

  • La couche zone_bassin-versant doit être en mode d’édition

  • Cliquez sur le bouton Connecter éléments réseau assainissement

  • Si vous déplacez la souris au-dessus d’un bassin versant, à proximité de sa périphérie, l’objet est mis en évidence: vous pouvez cliquer pour choisir cet objet.

  • Déplacez la souris sur l’élément du réseau auquel le bassin versant sera connecté (rappel: les objets sélectionnables sont mis en évidence) et cliquez à nouveau

  • Une fenêtre apparaît, dans laquelle vous pouvez choisir quelle(s) connexion(s) sera(ont) définie(s).

  • Si tout est OK, vous verrez apparaître un message dans une barre bleue, au-dessus de la vue de la carte

Note

Dans le modèle VSA-DSS, il est possible de connecter les bassins versants aux noeuds ou aux tronçons assainissement. Cependant, vu qu’un grand nombre de calculs hydrauliques ne supportent pas les connexions aux tronçons, cet outil dans QGEP se connecte uniquement aux noeuds assainissement et pas aux tronçons.

Note

L’outil ne contrôle pas si le noeud du réseau assainissement sélectionné fait partie du réseau pwwf (réseau principal). C’est exclusivement l’utilisateur qui décide si un noeud est correct ou pas. Si vous n’êtes pas sûr, consultez ** VSA Guide des données PGEE**, section Principes de saisie

Vous ne pouvez pas, en une seule opération avec l’outil, connecter de multiples zones avec un noeud assainissement. Si vous voulez le faire, vous devez sélectionner les bassins versants, ouvrir la table d’attributs et ensuite démarrer en mode d’édition multiple. Vous pouvez maintenant choisir le noeud assainissement à partir de la liste de tous les noeuds

Pour supprimer une connexion à un noeud assainissement, vous pouvez éditer la zone avec l’outil Identifier entités.
