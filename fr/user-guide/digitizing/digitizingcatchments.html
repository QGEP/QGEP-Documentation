2.1.7. Numérisation des Bassins Versants¶

2.1.7.1. Général¶

Les bassins versants peuvent être numérisés avec les outils QGIS standard.

Les bassins ne doivent pas se chevaucher. Pour cela, activez les Outils de numérisation avancés dans le menu Vue –> Barre d’outils

2.1.7.2. Numérisation¶

  • Passez la couche “od_catchment_area” en mode d’édition

  • Utilisez l’outil Ajouter une entité pour numériser un bassin versant.

  • Avant de commencer la numérisation, activez le bouton Enable Tracing

2.1.7.3. Édition¶

  • Vous pouvez éditez des bassins existants avec l’outil Sélectionner une entité.
