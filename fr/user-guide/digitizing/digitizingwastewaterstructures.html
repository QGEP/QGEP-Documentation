2.2.2. Numériser les structures assainissement¶

2.2.2.1. Général¶

Désactivez la case Réutiliser la dernière valeur attributaire saisie (Préférences -> Options -> Numérisation -> Création d’entités), ceci afin d’éviter que la saisie de votre deuxième objet soit bloquée. Ce bug sera corrigé au plus tard dans QGIS 3.0.

QGEP propose un outils d’assistance (wizard) pour construire les structures.

Sélectionner le bouton wizard puis cliquez sur le bouton Start Data Entry et choisissez Wastewater Structure dans le menu déroulant.

2.2.2.2. Numérisation¶

Le curseur passe alors en mode numérisation et il est possible de sélectionner l’emplacement sur la carte ou l’on veut créer l’élément.

Le formulaire vw_qgep_structure_assainissement s’ouvre ensuite et vous pouvez commencer la saisie de données dans l’onglet Général:

Sélectionnez le ws_type désiré (‘regard / trou d’homme’ est préselectionné):

  • Point d’exutoire

  • Installation d’infiltration

  • Regard de visite

  • Structure spéciale

Ajoutez ensuite l’identifiant (c’est l’attribut qui sera affiché sur la carte). Par défaut, l’identifiant de la structure assainissement est également l’identifiant du couvercle et du noeud assainissement.

Pour les chambres et les structures spéciales, choisissez la fonction dans l’attribut manhole_function ou special structure_function selon le type d’objet. Dans le future, seul l’attribut correspondant apparaîtra une fois le type ws_type choisit.

Ajoutez d’autres attributs dans l’onglet Général. Vous pouvez également ajouter des attributs dans les autres onglets (Couvercles, Structures assainissement, Regards et Noeuds assainissement). Cependant, vous devez sauvegarder les modifications apportées à structure assainissement dans la base de données avant de pouvoir créer de nouvelles donnés dans les onglets Couvercles, Structures assainissement, Noeuds assainissement ou Maintenance. Vous ne pouvez pas non plus utiliser Action pour les mêmes raisons. Sauvegardez tout d’abord l’enregistrement. Maintenant vous pouvez cliquer sur “”OK”.

Sauvegardez les informations de la couche en cliquant sur “Stop Data Entry” dans l’assistant (“wizard”)

Pour rééditer l’objet, passer en mode d’édition et cliquez sur l’objet à éditer avec le curseur en mode “Identifier les entités”. Si le mode d’édition n’est pas activé, vous pourrez seulement consulter les informations relatives à l’objet.

Pour plus d’informations au sujet de l’édition de données, voir le chapitre Modification de données existantes

2.2.2.3. Autres classes et attributs¶

Lorsqu’un objet point est numérisé, une série d’opérations se déroule en arrière-plan dans la base de données QGIS:

  1. Un nouvel objet est ajouté dans la classe wastewater_structure

  2. Un nouvel objet sera crée et lié dans les sous-classes respectives (point_de_déversement, installation_d_infiltration, chambre, structure_spéciale)

  3. un nouvel objet couvercle est ajouté et lié à la structure assainissement

  4. un nouvel objet noeud assainissement est généré dans la couche éléments du réseau assainissement et sa sous-classe noeuds assainissement

Pour ajouter des éléments additionnels comme des éléments de structure (en plus des couvercles, aide à l’accès, prévention contre débordement, etc.), vous devez d’abord sauver, puis éditer les objets ponctuels (voir le chapitre Modification de données existantes).
