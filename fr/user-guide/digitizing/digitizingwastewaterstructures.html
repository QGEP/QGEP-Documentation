2.1.2. Numériser les structures¶

2.1.2.1. Général¶

Disable the Reuse last entered attribute values option (Settings -> Options -> Digitizing) because it will stop your second new record. This bug will be corrected at least in QGIS 3.0.

QGEP propose un outils d’assistance (wizard) pour construire les structures.

Sélectionner le bouton Wizard puis cliquez sur le bouton Start Data Entry et choisissez Wastewater Structure dans le menu déroulant.

2.1.2.2. Numérisation¶

Le curseur passe alors en mode numérisation et il est possible de sélectionner l’emplacement sur la carte ou l’on veut créer l’élément.

Then the vw_qgep_wastewater_structure form opens and you can start adding data in the General tab:

Select the ws_type you want (preselected is ‘manhole’):

  • Point d’exutoire

  • Installation d’infiltration

  • Regard de visite

  • Structure spéciale

Then add the identifier (this is the attribute that will be displayed on the map). As default, the identifier of the wastewater structure is also the identifier of the cover and of the wastewater node.

Pour les chambres et les structures spéciales, choisissez la fonction dans l’attribut manhole_function ou special structure_function selon le type d’objet. Dans le future, seul l’attribut correspondant apparaîtra une fois le type ws_type choisit.

Add other attributes in the General tab. You can also add attributes in the other tabs (Cover, Wastewater Structure, Manhole, WasteWater Node) You can not add new records in the tab Covers, Structure Parts, Wastewater Nodes and Maintenance, because you have to save first the Wastewater Structure-Record to the database before you can add aditional records. You can not use Aktionen for the same reason. First safe the record. Now you can click OK.

Sauvegarder les informations de la couche en stoppant le “wizard”

Pour réediter l’objet, passer en mode d’édition et cliquez sur l’objet avec le curseur information. Si le mode d’édition n’est pas activé, vous pourrez seulement consulter les informations relatives à l’objet.

Pour de plus d’informations à propos de l’édition de données, voir le chapitre Modification de données existantes

2.1.2.3. Autres classes et attributs¶

When a point object is digitized a series of steps take place in the background in the QGIS database:

  1. an new object in the class wastewater structure is added
  2. a new object in the respective subclass [discharge_point, infiltration_installation, manhole, special_structure] is added and linked
  3. a new cover object is added and linked to the waste water structure
  4. a new wastewater node object is generated in wastewater network elements and its subclass wastewater nodes

To add additional objects such as structure parts (besides covers there are access aids, backflow prevention etc) you have to save first and then edit the point object (see the Modification de données existantes chapter).

..(take more from GEPView-Howto 1.2. Chapter 4.1)
