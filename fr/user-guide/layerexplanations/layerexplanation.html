2.1.2. Description de la couche¶

2.1.2.1. Couches principales¶

2.1.2.1.1. vw_qgep_structure_assainissement¶

Couche principale pour les chambres / regards de visite, structures spéciales, installations d’infiltration, points de déversement (et stations d’épuration des eaux usées - Step). La création d’un nouvel enregistrement dans cette couche génère toujours un nouveau couvercle, un nouveau noeud assainissement et une nouvelle structure assainissement. Depuis le formulaire d’édition, vous avez accès à de nombreuses tables liées (p.ex. pièces de structure, événements de maintenance).

2.1.2.1.2. vw_qgep_tronçon¶

Couche principale pour les structures d’assainissement de type linéaire (conduites). La création d’un nouvel enregistrement dans cette couche génère toujours un nouveau tronçon (reach) et une nouvelle conduite (channel). Depuis le formulaire d’édition, vous avez accès à de nombreuses tables liées (p.ex. pièces de structure, événements de maintenance).

2.1.2.2. Structures assainissement¶

2.1.2.2.1. od_structure_assainissement¶

Cette couche affiche les géométries détaillées des structures assainissement. Vous numérisez une nouvelle géométrie détaillée par des enregistrements dans la couche vw_qgep_structure_assainissement. Si vous souhaitez modifier une géométrie existante, vous devez alors l’éditer dans la couche od_structure_assainissement.

2.1.2.2.2. Pièces de structure¶

Le couvercle est le seul élément d’une structure assainissement avec une géométrie ponctuelle propre. Tous les autres éléments ne sont reliés que par un lien à leur structures assainissement respectives et devraient par conséquent être édités uniquement à partir des couches principales (vw_qgep_structure_assainissement et vw_qgep_troncon).

2.1.2.2.3. vw_couvercle¶

Utiliser cette couche pour modifier la géométrie du couvercle sélectionné uniquement (et non la structure assainissement complète), ou pour ajouter un nouveau couvercle à une structure assainissement existante. (Vous pouvez également ajouter un couvercle supplémentaire via l’onglet Couvercle du formulaire vw_qgep_structure_assainissement, mais ce couvercle n’aura pas de géométrie...).

2.1.2.2.4. vw_tronçon¶

Cette couche a-t-elle une utilité ou peut-elle être supprimée ?

2.1.2.2.5. od_organisation¶

A FAIRE: Cette table se trouve actuellement à un endroit inapproprié et doit être révisée.

2.1.2.3. Inspection¶

Ces tables se trouvent dans les onglets Maintenance des tables principales. Ne pas éditer directement.

2.1.2.4. Listes de valeurs¶

Ces listes de valeurs sont définies dans le modèle de données VSA. Ne pas les modifier.

2.1.2.5. Hydraulique / vw_noeud_assainissement¶

Utilisez cette couche pour modifier la géométrie du noeud sélectionné uniquement (et non la structure assainissement complète) ou pour ajouter un nouveau noeud à une structure assainissement existante. (Il est également possible de créer un noeud supplémentaire via l’onglet Noeud du formulaire vw_qgep_structure_assainissement, mais ce noeud n’aura pas de géométrie...).

Cette couche doit être développée dans le futur avec des tables liées (p.ex. des tables débordement...)

2.1.2.6. Topologie¶

Ces deux couches sont utilisées par le plugin QGEP pour les modules Profil en long et Poursuite de réseau. Ne pas modifier ou utiliser pour un autre usage.
