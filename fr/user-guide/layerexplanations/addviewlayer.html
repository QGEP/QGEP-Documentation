Ajout d’une couche vue

  • Ajouter une couche basée sur une vue dans la boîte de dialogue Ajouter une couche va générer le message suivant, lorsque vous sélectionnez une vue

  • Pour éviter cela, sélectionnez la couche « vue » puis faire défiler la colonne « feature id » et sélectionnez « obj_id » comme élément pertinent, car QGIS ne peut pas le reconnaître automatiquement.
