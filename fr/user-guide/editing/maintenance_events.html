2.3.3. Ajout et édition d’événements de maintenance¶

Guide pour l’édition ou l’ajout de données relatives aux événements de maintenance dans QGEP.

2.3.3.1. Général¶

  • Les événements de maintenance peuvent être connectés à une ou plusieurs structures. Pour cela, QGEP utilise une table intermédiaire RE_table afin de generé la relation n:m (re_maintenance_event_wastewater_structure)

2.3.3.2. Ajout (d’un seul événement appartenant à une seule structure)¶

  • Activez le mode d’édition.

  • Avec l’outil Identifier les entités, vous pouvez sélectionner des chambres ou des tronçons en sélectionnant leur couche respective (vw_qgep_couvercle ou vw_qgep_tronçon).

  • Le formulaire correspondant va s’ouvrir. Allez dans l’onglet Evénement de maintenance.

  • Créez un nouvel événement de maintenance

  • Ou relier la structure avec un événement de maintenance existant

  • Fermez le formulaire avec OK et désactivez le mode d’édition pour sauvegarder les changements
