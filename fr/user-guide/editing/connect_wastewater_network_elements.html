2.3.4. Connexion des éléments du réseau assainissement¶

Procédure pour connecter les éléments du réseau assainissement dans QGEP.

2.3.4.1. Généralités¶

  • QGEP dispose d’un outil permettant de connecter des éléments du réseau assainissement

2.3.4.2. Comment connecter les tronçons aux noeuds ou les noeuds à d’autres noeuds¶

  • Peu importe la couche sélectionnée

  • La couche vw_qgep_troncon doit être en mode d’édition

  • Cliquez sur le bouton Connecter les éléments du réseau assainissement

  • Si vous passez avec la souris au-dessus d’un tronçon qui peut être connecté, le message “vous pouvez cliquer” s’affiche.

  • Déplacez la souris sur l’élément du réseau auquel le premier objet doit être connecté (rappel: les objets sélectionnables sont surlignés) et cliquez à nouveau

  • Une fenêtre apparaît, dans laquelle cous pouvez choisir quelle connexion sera définie. Si vous connectez un tronçon, la fenêtre aura cet aspect:

  • Si tout est OK, vous verrez apparaître un message dans une barre bleue, au-dessus de la vue de la carte

2.3.4.4. Comment supprimer une connection¶

Pour l’instant, il n’y a pas d’outil spécial permettant de supprimer une connexion entre éléments du réseau.

  • Sélectionnez la couche vw_qgep_reach et basculez en mode d’édition

  • Cliquez sur le bouton Identifier objets et sélectionnez le tronçon à modifier par double-clic

  • Supprimez la valeur dans le champ rp_…_fk_wastewater_networkelement pour supprimer la connexion

Note

Soyez conscient des principes de saisie selon le VSA : Un point de tronçon « depuis » (rp_from_fk_wastewater_networkelement) ne devrait jamais être connecté à un autre tronçon.

Par exemple, lorsque un tronçon se déverse directement dans un autre sans structure entre les deux, le tronçon amont devrait être connecté au tronçon aval (rp_from_fk_wastewater_networkelement = downstream reach). Inversement, le tronçon aval ne devrait pas être connecté avec celui en amont. Cette règle s’applique pour les tronçons swwf (réseau secondaire). Pour le réseau pwwf (réseau principal) par contre, il doit toujours y avoir un noeud entre deux tronçons.
