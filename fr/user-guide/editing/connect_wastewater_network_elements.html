2.3.4. Connexion des éléments du réseau assainissement¶

Procédure pour connecter les éléments du réseau assainissement dans QGEP.

2.3.4.1. Général¶

  • QGEP dispose d’un outil permettant de connecter des éléments du réseau assainissement

2.3.4.2. Comment connecter les tronçons aux noeuds ou les noeuds à d’autres noeuds¶

  • Peu importe la couche sélectionnée

  • La couche vw_qgep_troncon doit être en mode édition

  • Cliquer sur le bouton Connecter éléments réseau assainissement

  • If you move the mouse over an reach, that can be connected, it is marked: you can click
  • Move the mouse to the networkelement, to which the first object will by connected (again: selectable objects get marked) and click again
  • A window appears, where you can choose, which connection will be defined. If you connect a reach, the window looks like this:

  • Si tout est OK, vous verrez apparaître un message dans une barre bleue, au-dessus de la vue de la carte

2.3.4.4. Comment supprimer une connection¶

Il n’existe pas pour le moment d’outil spécial pour supprimer une connexion entre éléments du réseau.

  • Select the vw_qgep_reach layer and set it to edit mode

  • Cliquer sur le bouton Identifier objets et sélectionner le tronçon à modifier par double-clic

  • Delete the value in the rp_..._fk_wastewater_networkelement field to delete the connection

Note

Be aware of the ‘VSA Erfassungsgrundsätze’: A reach point from (rp_from_fk_wastewater_networkelement) should never be connected to another reach.

For example, when you have a reach discharging directly into another one without a structure between the two, the upstream reach should be connected to the downstream reach (rp_from_fk_wastewater_networkelement = downstream reach). But inversely, the downstream reach shouldn’t be connected to the upstream one. This rule is for swwf-channels (secondary network). In pwwf-network, there must be always a node between to reaches.
