5.1. Introduction¶

Wincan 2 QGEP est un plugin QGIS pour l’import des données d’inspection Wincan dans QGEP. A l’heure actuelle, il fonctionne uniquement avec le format Wincan 8, mais une mise à niveau est prévue pour supporter Wincan VX et/ou le format pur VSA-DSS KEK.

Ce plugin peut être décrit comme une procédure de mise en correspondance entre les données d’inspection TV et les données du réseau.

Le plugin va lire le fichier XML à partir des données et offrir une interface pour rechercher et sélectionner sur la carte les conduites correspondantes à celles provenant des données d’inspection. Une fois cette correspondance établie, les données pourront être importées dans QGEP.
