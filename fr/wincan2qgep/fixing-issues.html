5.4. Résoudre les problèmes¶

5.4.1. Problèmes de longueur¶

Si la longueur de la conduite est plus grande dans les données que sur la carte, un avertissement va apparaître. Cela peut être corrigé dans les données ou être contraint en cochant la colonne correspondante dans la table observations.

5.4.2. Une conduite dans les données d’inspection correspond à plusieurs conduites dans les données QGEP¶

Il est possible qu’une conduite dans les données d’inspection corresponde à plusieurs conduites dans les données QGEP. Pour corriger cela, vous pouvez sélectionner jusqu’à 3 conduites dans les widgets. Les longueurs d’inspection seront modifiées en conséquence.

5.4.3. Plusieurs conduites dans les données d’inspection correspondent à une seule conduite dans les données QGEP¶

Il est possible que plusieurs conduites dans les données d’inspection correspondent à une seule conduite dans les données QGEP. Pour corriger cela, sélectionnez la boîte à cocher « Réutiliser conduite précédente ». Les longueurs d’inspection seront modifiées en conséquence.
