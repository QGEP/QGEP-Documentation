6.3. Faire correspondre les conduites

Afin d’importer les données de l’inspection, les conduites inspectées doivent être mises en correspondance avec les conduites QGEP. Cette tâche peut être effectuée de manière manuelle ou semi-automatique. Tant que les conduites ne sont pas mises en correspondance avec les données existantes, elles sont surlignées en rouge dans la table principale.

6.3.1. Correspondance automatique

Une correspondance automatique peut être effectuée en recherchant une suite de caractéristique dans les identifiants du regard. Entrez un identifiant de conduite dans le champ correspondant et le plugin va rechercher les conduites connectées à ce regard de la manière suivante: {channel_id} - {manhole_id} (l’identifiant du regard est repris des données d’inspection).

6.3.2. Correspondance manuelle

La correspondance manuelle peut être réalisée de la façon suivante:

  1. Sélectionnez la conduite désirée dans la table

  2. Dans le premier widget de la conduite, cliquez sur le bouton Sélectionner dans la carte

  1. Trouvez et sélectionnez la conduite correspondante dans la carte.

  2. L’identifiant de la conduite est maintenant listé et la ligne n’est plus surlignée dans la table.

6.3.3. Ignorer des conduites ou des observations

Si vous souhaitez ignorer certaines des conduites inspectées, décochez-les simplement dans la table conduites. Procédez de la même manière pour les observations.
