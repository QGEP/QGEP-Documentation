6.2. Ouvrir une inspection

Depuis le menu du plugin (Plugins > Wincan ou à partir de la barre d’outils du plugin, sélectionner Ouvrir une inspection), sélectionner le fichier XML généralement situé dans XML > Project.xml.

Vous devez spécifier au préalable l’emplacement du document de report en le définissant dans le champ PDF. Ceci permettra de le lier dans le formulaire d’entité.

Les conduites sont répertoriées en rouge dans la table principale car elles doivent être liées avec les données QGEP avant d’être importées.
