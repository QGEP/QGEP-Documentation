3.1.2. Utiliser pgAdmin¶

3.1.2.1. Requêtes SQL¶

Exécuter des commandes SQL dans PgAdmin III

Note

  • En cliquant sur l’icône mis en évidence par le point 1 vous pouvez ouvrir une requête SQL. L’icône SQL n’est actif que lorsque vous êtes connecté à une base de données.

  • En cliquant sur l’icône mis en évidence par le point 2 permet de lancer la requête SQL.

3.1.2.2. “Dump” de la base de données¶

Note

  • Ouvrir pgAdmin III et se connecter à la base de données

  • Faire un clic droit sur le schéma (dans l’arborescence sous “Schemas”) que l’on veut sauvegarder

  • Entrer l’endroit ou le fichier devra être sauvegardé

  • Cocher les options suivantes

Note

  • Après avoir fait tourner le processus de sauvegarde, le code de sortie devrait indiquer 0 si tout s’est bien passé
