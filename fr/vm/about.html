5.1. À propos

Guide pour télécharger et installer une machine virtuelle (MV) incluant une installation fonctionnelle de QGEP avec un projet de démonstration. Cette MV inclut également une démo fonctionnelle de QWAT water project

La VM est basée sur Debian Testing avec l’environnement de bureau “Cinnamon”.

Debian Testing a été retenu dans le but de disposer de GDAL 2.x et de bénéficier également des derniers noyaux Linux, afin que les hardware les plus récents ne soient pas un problème dans le cas d’une MV utilisée hors virtualisation (p. ex. pour le cas où un disque virtuel est utilisé comme support pour créer une clé usb physique à utiliser à des fins de démonstration).

Logiciels installés:

  • PostgreSQL 9.6.0 avec Postgis 2.3

  • QGIS 2.18

  • pgAdmin 4 (open the browser at http://localhost:5050 to access it)

Paramètres différents d’une installation logicielle par défaut:

  • pg_hba.conf has trust for local connections

  • QGIS est installé avec multithreaded rendering enabled

Note

Les utilisateurs qgis et root ont le mot de passe qgis

Note

Si vous décidez d’utiliser la MV en production, il est recommandé de:

  • Changer les mots de passe

  • regenerate the ssh keys by running in a terminal sudo rm /etc/ssh/ssh_host_* && sudo dpkg-reconfigure openssh-server
